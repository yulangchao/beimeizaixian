邮箱资料配置>>
邮件发送模式: SMTP方式发送(常用) mail 函数发送(极少用,绝大部分服务器都不支持)
SMTP 服务器: 如:个人QQ邮箱smtp.qq.com 或者是 企业QQ邮箱smtp.exmail.qq.com 或是 smtp.163.com
邮箱帐号: 如:test@qq.com
邮箱密码（QQ授权码）: 注意：使用QQ邮箱的话，请输入授权码，而不是邮箱登录密码
SMTP 端口: 25或465，QQ邮箱是465
[测试发邮件]
说明
  • 1、一般情况下都是选择“SMTP 模块发送”。而“mail 函数发送”只有极少部分服务器才支持，windows服务器一般不支持。
  • 2、选择SMTP方式发送邮件,一般推荐使用QQ邮箱,或者163邮箱.关于QQ邮箱的设置,请点击查看帮助文档
  • 3、关于检查25端口是否被关闭的方法，请把引号内的代码“<?php print_r(fsockopen('smtp.qq.com', 25, \$errno, \$errstr, 5));?>”另存为test.php文件放到网站根目录访问，如能看到"Resource id #2"这样的字眼，则25端口没关闭，否则就被关闭了。
